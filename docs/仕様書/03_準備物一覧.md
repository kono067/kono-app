# 議事録ジェネレーター 準備物一覧

ユーザー（あなた）が事前に用意する必要があるものの一覧です。
具体的な操作方法がわからない場合は、GPT に各項目の名前をそのまま聞いてください。

## 1. Google OAuth 認証情報（Googleログイン用）

Googleログインを実装するために必要です。

### 取得するもの
| 項目 | 説明 |
|------|------|
| **Google Client ID** | `xxx.apps.googleusercontent.com` の形式 |
| **Google Client Secret** | 英数字の文字列 |

### 取得場所
[Google Cloud Console](https://console.cloud.google.com/)

### GPTに聞くときの質問例
> 「Google Cloud Console で OAuth 2.0 クライアントIDを作成する方法を教えてください。リダイレクトURIは `http://localhost:5173/api/auth/callback/google` です」

### 設定時の注意点
- **承認済みのリダイレクトURI** に以下の2つを追加してください：
  - 開発用: `http://localhost:5173/api/auth/callback/google`
  - 本番用: `https://あなたのドメイン/api/auth/callback/google`（デプロイ後に追加）

## 2. Turso データベース（データ保存用）

ユーザー情報、クレジット、議事録を保存するデータベースです。

### 取得するもの
| 項目 | 説明 |
|------|------|
| **Database URL** | `libsql://xxx.turso.io` の形式 |
| **Auth Token** | 英数字の長い文字列 |

### 取得場所
[Turso ダッシュボード](https://turso.tech/)

### GPTに聞くときの質問例
> 「Turso でデータベースを作成して、Database URL と Auth Token を取得する方法を教えてください」

### 手順の概要
1. Turso にサインアップ（GitHub アカウントで可能）
2. 新しいデータベースを作成（名前は `minutes-generator` など）
3. Dashboard から Database URL と Auth Token をコピー

## 3. Polar アカウント（決済用）

サブスクリプション課金を処理するための決済サービスです。

### 取得するもの
| 項目 | 説明 |
|------|------|
| **Polar Access Token** | 組織のアクセストークン |
| **Polar Webhook Secret** | Webhook の署名検証用シークレット |
| **梅プランの Product ID** | Polar で作成した商品のID |
| **竹プランの Product ID** | Polar で作成した商品のID |
| **松プランの Product ID** | Polar で作成した商品のID |

### 取得場所
[Polar ダッシュボード](https://polar.sh/) （テスト用は [Sandbox](https://sandbox.polar.sh/)）

### GPTに聞くときの質問例
> 「Polar.sh で組織を作成し、3つのサブスクリプション商品（月額¥500、¥1,280、¥2,980）を作成して、Access Token と Webhook Secret を取得する方法を教えてください」

### 手順の概要
1. Polar にサインアップ
2. 組織（Organization）を作成
3. **まず Sandbox 環境（sandbox.polar.sh）でテスト**
4. 3つのサブスクリプション商品を作成：
   - 梅プラン: ¥500/月
   - 竹プラン: ¥1,280/月
   - 松プラン: ¥2,980/月
5. Settings > Access Tokens でアクセストークンを発行
6. Settings > Webhooks で Webhook エンドポイントを追加
   - URL: `https://あなたのドメイン/api/webhooks/polar`（デプロイ後に設定）
   - イベント: `order.created`, `subscription.created`, `subscription.updated`, `subscription.canceled`

## 6. 環境変数のまとめ

最終的に `.env` ファイルに設定する変数の一覧です。

```env
# ✅ 取得済み
GEMINI_API_KEY=（現在のAPIキー）

# 📋 要取得: Google OAuth
GOOGLE_CLIENT_ID=（Google Cloud Console で取得）
GOOGLE_CLIENT_SECRET=（Google Cloud Console で取得）

# 📋 要取得: Turso
TURSO_DATABASE_URL=（Turso で取得）
TURSO_AUTH_TOKEN=（Turso で取得）

# 📋 要取得: Polar
POLAR_ACCESS_TOKEN=（Polar で取得）
POLAR_WEBHOOK_SECRET=（Polar で取得）
POLAR_PRODUCT_UME=（Polar 梅プランの Product ID）
POLAR_PRODUCT_TAKE=（Polar 竹プランの Product ID）
POLAR_PRODUCT_MATSU=（Polar 松プランの Product ID）

# 🔧 自動生成（開発者が設定）
BETTER_AUTH_SECRET=（ランダムな文字列 - 開発者が生成）
BETTER_AUTH_URL=http://localhost:5173
```
