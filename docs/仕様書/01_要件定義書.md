# 議事録ジェネレーター 要件定義書

## 1. アプリ概要

**アプリ名**: （未定 - 後日決定）

会議中に取った雑なメモを入力すると、指定されたテンプレート形式の議事録を自動生成するWebアプリ。

### 対象ユーザー
- 営業・企画・人事などの非エンジニア職
- 会議中に簡単な箇条書きや断片的なメモを取ることが多い
- メモを議事録として整理する作業に時間がかかっている

### 解決する課題
会議メモから議事録を作成する手作業をAIで自動化し、30秒以内で完了させる。

---

## 2. 機能一覧

### コア機能
| # | 機能 | 説明 |
|---|------|------|
| F1 | メモ入力 | 会議メモをテキストエリアに入力/貼り付け |
| F2 | テンプレート選択 | ドロップダウンから固定テンプレートを選択 |
| F3 | 議事録生成 | AIがメモをテンプレート形式に整理して議事録を生成 |
| F4 | 議事録コピー | 生成された議事録をクリップボードにコピー |
| F5 | 議事録保存 | 生成した議事録をアカウントに紐づけて保存 |
| F6 | 議事録履歴 | 過去に生成した議事録の一覧表示・再表示 |

### 認証機能
| # | 機能 | 説明 |
|---|------|------|
| A1 | Googleログイン | Better Auth を使用したGoogleログイン |
| A2 | セッション管理 | ログイン状態の保持・自動ログアウト |

### 課金・クレジット機能
| # | 機能 | 説明 |
|---|------|------|
| C1 | 初回無料クレジット | 初回ログイン時に5回分のクレジットを付与 |
| C2 | クレジット消費 | 議事録生成1回につき1クレジット消費 |
| C3 | 残りクレジット表示 | 現在のクレジット残量を画面に表示 |
| C4 | プラン選択・課金 | Polar を使用した3段階のサブスクリプション課金 |
| C5 | クレジット不足時の案内 | クレジット切れ時にプラン選択を促す |

### 実装しない機能
- 音声録音
- 会議の自動文字起こし
- テンプレート作成・カスタマイズ機能
- 外部サービス連携（Slack / Notion など）
- 議事録の編集・再生成

## 3. ユーザー利用フロー

### 初回利用
1. トップページにアクセス
2. Googleアカウントでログイン
3. 初回無料クレジット（5回分）が自動付与される
4. メモを入力 → テンプレート選択 → 議事録生成

### 通常利用
1. ログイン状態でアクセス
2. メモをテキストエリアに入力/貼り付け
3. テンプレートを選択（ドロップダウン）
4. 「議事録を生成」ボタンを押す
5. AIが議事録を生成して表示
6. 「コピー」ボタンで議事録をコピー
7. 議事録は自動的に履歴に保存される

### クレジット切れ時
1. 議事録生成ボタンを押す
2. クレジット不足のメッセージが表示される
3. プラン選択画面に誘導される
4. Polar の決済画面でプランを購入
5. クレジットが付与され、利用を再開

## 4. 入力・出力仕様

### 入力
ユーザーが会議中に取った雑なメモ（テキスト）。
箇条書き、断片的な文章、キーワードの羅列でも可。

**入力例:**
```
・A社 見積 来週提出
・UI 修正必要
・山田 テスト確認
・来週火曜 再確認
```

### 出力
選択したテンプレートに沿った構造化された議事録。

**出力例:**
```
■決定事項
A社への見積は来週提出する

■課題
UIの修正が必要

■アクション
山田がテスト確認を行う

■次回確認
来週火曜日に進捗を再確認する
```

---

## 5. テンプレート定義

### テンプレート1: 社内会議
```
■議題
■決定事項
■課題
■次回アクション
```

### テンプレート2: 営業ミーティング
```
■顧客要望
■決定事項
■課題
■次のアクション
```

### テンプレート3: 1on1ミーティング
```
■話した内容
■課題
■次回アクション
```

## 6. 課金プラン

### 無料枠
- 初回ログイン時に **5回分** のクレジットを付与
- クレジットに有効期限なし

### サブスクリプションプラン

| プラン | 月額（税込） | 議事録生成回数 | 想定ユーザー |
|--------|-------------|---------------|-------------|
| 🌸 梅プラン（ライト） | ¥500/月 | 30回/月 | 週1〜2回の会議がある人 |
| 🎋 竹プラン（スタンダード） | ¥1,280/月 | 100回/月 | 毎日会議がある人 |
| 🌲 松プラン（プレミアム） | ¥2,980/月 | 無制限 | チームで大量に使う人 |

### クレジットの仕組み
- サブスクリプション購入時に月間クレジットが付与される
- 毎月の更新日にクレジットがリセット・再付与される
- 松プランは無制限（クレジットチェックをスキップ）
- 未使用クレジットは翌月に繰り越さない

---

## 7. 非機能要件

### パフォーマンス
- 議事録生成は 30秒以内 に完了すること
- ページの初期読み込みは 3秒以内

### セキュリティ
- APIキーはサーバーサイドで管理（フロントに露出させない）
- 認証にはBetter Authを使用
- HTTPS通信のみ

### 対応環境
- PC ブラウザ（Chrome, Safari, Firefox, Edge）
- モバイルブラウザ（レスポンシブ対応）
